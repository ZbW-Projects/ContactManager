# ContactManager

## Installation

Die aktuelle Version kann auf der [Releases-Seite](https://github.com/ZbW-Projects/ContactManager/releases/tag/JEN25.09.21) heruntergeladen werden.

### 1. Eigenständige EXE-Datei
- **Datei:** `ContactManager.exe`  
- **Beschreibung:** Selbstenthaltende Windows-Executable.  
- **Wichtig:** Diese Version enthält **keine** Demo- oder Testdaten.  
- **Einsatz:** Empfohlen, wenn du eine saubere Installation möchtest und deine eigenen Daten anlegen oder importieren willst.

### 2. Vorgepacktes ZIP
- **Datei:** `ContactManager.zip`  
- **Beschreibung:** Enthält die ausführbare Datei **sowie** mitgelieferte Demo-/Testdaten.  
- **Einsatz:** Empfohlen, wenn du die Anwendung schnell mit Beispieldaten ausprobieren möchtest.

### 3. Quellcode-Archive
- **Dateien:**  
  - `ContactManager-<version>.zip`  
  - `ContactManager-<version>.tar.gz`  
- **Beschreibung:** Automatisch erzeugte Quellcode-Archive des Projekts.  
- **Einsatz:** Nützlich, wenn du den Code einsehen oder das Projekt selbst bauen möchtest.

---

## Schnellstart
1. Lade die gewünschte Datei aus den Release-Assets herunter.  
2. Entpacke sie bei Bedarf (`ContactManager.zip` oder Quellcode-Archive).  
3. Starte `ContactManager.exe`.  

---

## Hinweise
- Die Anwendung benötigt **keine zusätzliche Installation** – einfach herunterladen und starten.  
- Demo-/Testdaten sind nur im ZIP-Paket enthalten.  
- Für Entwickler: Ein Build aus dem Quellcode erfordert [.NET 9.0](https://dotnet.microsoft.com/).
